## Butler Automation

### Automation Capabilities

:::note Butler Power
Butler is Trello's built-in automation tool that helps teams work more efficiently by automating repetitive tasks.
:::

1. **Rule Types**

   - Card actions
   - Board actions
   - Due date management
   - List operations

2. **Trigger Events**
   - Card moves
   - Due dates
   - Label changes
   - Member actions

## Flow Metrics

### Lead Time & Cycle Time

```mermaid
graph LR
    A[Customer Request] -->|Lead Time| B[Delivery]
    C[Work Start] -->|Cycle Time| B
    A -->|Wait Time| C
```

<Tabs>
  <TabItem value="lead" label="Lead Time" default>
    **Definition**:
    - Total time from request to delivery
    - Includes all waiting periods
    - Customer perspective
    - End-to-end measurement

    **Measurement**:
    - Start: Request received
    - End: Delivery complete
    - Include all states
    - Track by service class

  </TabItem>
  <TabItem value="cycle" label="Cycle Time">
    **Definition**:
    - Active work time
    - Process efficiency
    - Team perspective
    - Value-add time

    **Measurement**:
    - Start: Work begins
    - End: Work complete
    - Exclude waiting
    - Track by work type

  </TabItem>
</Tabs>

### Flow Efficiency

:::info Flow Efficiency Formula
Flow Efficiency = (Value-Adding Time / Total Lead Time) Ã— 100%
:::

1. **Components**

   - Value-adding time
   - Non-value-adding time
   - Wait time
   - Block time

2. **Improvement Focus**
   - Reduce wait states
   - Minimize handoffs
   - Optimize processes
   - Remove blockers
